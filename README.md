## 📅 일정

<br>

## 📌 목표

1. 데이터 분석
2. 웹 페이지 제작

<br>

## 📖 상세 내용

- 4대 질병 시계열 시각화 및 주기성 분석 및 해석
- 지역별 특정 질병이 발생하는지 분석
- 건강검진 데이터를 활용하여 어떤 질병에 취약한지 분류분석 머신러닝
- 위의 분석내용을 웹서비스로 구현
